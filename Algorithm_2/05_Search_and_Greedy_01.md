# 반복(Iteration)과 재귀(Recursion)

- 2개는 비슷한 작업을 할 수 있음
- 반복은 수행하는 작업이 완료될 때까지 계속 반복

 - 루프(for, while 구조)

 - 반복문은 코드를 n번 반복시킬 수 있음

- 재귀는 주어진 문제의 해를 구하기 위해 동일하면서 더 작은 문제의 해를 이용하는 방법

 - 하나의 큰 문제를 해결할 수 있는 (해결하기 쉬운) 더 작은 문제로 쪼개고 결과들을 결합

 - 재귀 호출은 n중 반복문을 만들어낼 수 있다
