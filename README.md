# TIL

## MarkDown
- 일반 텍스토로 문서를 작성하는 간단한 방법
- #의 개수에 따라 제목의 수준을 구별

### 리스트
- 목록을 표시하기 위해 사용
- 순서가 있는 리스트와 없는 리스트 제공

### ```
- 일반 텍스트와 달리 해당 프로그래밍 언어에 맞춰 텍스트 스타일을 변환

### 링크(Link)&이미지(Image)
- 특정 주소를 사용해 다른 페이지로 이동하는 링크 혹은 이미지를 출력
- 너비와 높이는 마크다운으로 조절할 수 없음

### 텍스트 관련
**굵게**
*기울임*
~~ 취소선 ~~

### 수평선
- 단락을 구분할 때 사용
- '-' 3개 이상 적으면 작동
